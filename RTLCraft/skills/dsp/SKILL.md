# dsp — Digital Signal Processor Skill

## Overview

DSP suite using the Spec2RTL flow.
Signed multipliers, I/Q synchronizers, I2S audio interface,
DDS sine/cosine generator, and CIC sample-rate converters.

Reference RTL: `ref_rtl/dsp/rtl/` (Alex Forencich open-source DSP library).

## Modules

| File | Description |
|------|-------------|
| `behaviors.py` | 12 behavior templates (dsp_mult, iq_join, iq_split, i2s_ctrl, phase_accumulator, dsp_iq_mult, i2s_rx, i2s_tx, sine_dds_lut, sine_dds, cic_decimator, cic_interpolator) |
| `models.py` | Golden reference models for all 12 DSP modules |
| `arch_templates.py` | build_dsp_arch(), DSP_SuiteModel |
| `skeleton_templates.py` | PE type → implementation steps (12 PE types) |

## Quick Start

### Build Architecture

```python
from skills.dsp.arch_templates import build_dsp_arch

arch = build_dsp_arch(
    mult_width=16,
    phase_width=32,
    dds_output_width=16,
    cic_width=16,
    cic_rmax=2,
    cic_m=1,
    cic_n=2,
    i2s_width=16,
)
```

### Golden Reference Verification

```python
from skills.dsp.models import (
    DSP_MULT_Model, SINE_DDS_Model,
    CIC_DECIMATOR_Model, CIC_INTERPOLATOR_Model,
    I2S_RX_Model, I2S_TX_Model,
)

# Multiplier
mult = DSP_MULT_Model(width=16)
for _ in range(10):
    a_tready, b_tready, out_data, out_valid = mult.step(
        input_a_tdata=100, input_b_tdata=200,
        input_a_tvalid=1, input_b_tvalid=1,
        output_tready=1,
    )

# DDS sine/cosine generator
dds = SINE_DDS_Model(phase_width=32, output_width=16)
for _ in range(1000):
    _, _, sample_i, sample_q, valid = dds.step(
        input_phase_tvalid=0, input_phase_step_tvalid=0,
        output_sample_tready=1,
    )

# CIC decimator
cic = CIC_DECIMATOR_Model(width=16, rmax=2, n=2)
for i in range(100):
    tready, tdata, tvalid = cic.step(
        input_tdata=i, input_tvalid=1,
        output_tready=1, rate=2,
    )
```

## DSP Pipeline Stages

| Stage | pe_type | Description | Latency |
|-------|---------|-------------|---------|
| DSP_MULT | dsp_mult | 4-stage signed scalar multiplier | 4 cycles |
| IQ_JOIN | iq_join | I/Q channel synchronizer | Variable |
| IQ_SPLIT | iq_split | I/Q channel demultiplexer | Variable |
| I2S_CTRL | i2s_ctrl | I2S clock generator (prescaler) | Free-running |
| PHASE_ACCUMULATOR | phase_accumulator | NCO phase ramp generator | 1 cycle |
| DSP_IQ_MULT | dsp_iq_mult | Complex IQ multiplier (I×I, Q×Q) | 4 cycles |
| I2S_RX | i2s_rx | I2S serial receiver (edge-detect sck) | Variable |
| I2S_TX | i2s_tx | I2S serial transmitter (dual-edge sck) | Variable |
| SINE_DDS_LUT | sine_dds_lut | Sine/cosine LUT (fine/coarse decomposition) | 5 cycles |
| SINE_DDS | sine_dds | Full DDS (phase_accum + LUT) | 5+ cycles |
| CIC_DECIMATOR | cic_decimator | CIC decimator (integrator→decimate→comb) | Variable |
| CIC_INTERPOLATOR | cic_interpolator | CIC interpolator (comb→upconvert→integrator) | Variable |

## Key Algorithms

### 4-Stage Pipeline (DSP_MULT, DSP_IQ_MULT)

Maps to Xilinx DSP slices:

```
Cycle 0: input_reg_0 ← input data
Cycle 1: input_reg_1 ← input_reg_0 (pipeline register)
Cycle 2: output_reg_0 ← multiply (DSP48 slice)
Cycle 3: output_reg_1 ← output_reg_0 (pipeline register)
```

AXI-Stream backpressure: `input_a_tready = input_b_tvalid & output_tready`

### Fine/Coarse Angle Decomposition (SINE_DDS_LUT)

Reduces ROM size using angle-addition identity:

```
sin(A+B) = sin(A) + cos(A) * sin(B)
cos(A+B) = cos(A) - sin(A) * sin(B)
```

Where:
- A = coarse angle (top bits of phase, W+1 bits)
- B = fine angle (bottom bits of phase, W bits)
- coarse_c/s_lut: 2^(W+1) entries (cos/sin of coarse angles)
- fine_s_lut: 2^W entries (sin of fine angles only, smaller table)

5-stage pipeline:
1. LUT read (coarse_c, coarse_s, fine_s)
2. Pipeline register
3. Multiply (cos*fine, sin*fine)
4. Add/subtract with right-shift normalization
5. Sign correction (SIGN bit negates output)

### CIC Filter Structure

**Decimator** (sample-rate reduction):
```
IN → N Integrators (z⁻¹ feedback) → Decimate (every R cycles) → N Combs (z⁻ᴹ feedforward) → OUT
```

**Interpolator** (sample-rate increase):
```
IN → N Combs (z⁻ᴹ feedforward) → Up-convert (insert zeros) → N Integrators (z⁻¹ feedback) → OUT
```

Parameters:
- **N**: Filter order (number of stages)
- **M**: Differential delay
- **R**: Decimation/interpolation ratio (programmable)
- **REG_WIDTH**: WIDTH + ceil(log₂((R×M)^N)) for growth bits

### I2S Protocol

- **Clock**: sck generated by prescaler from system clock
- **Word Select**: ws toggles every WIDTH sck cycles (left=0, right=1)
- **RX**: Edge-detects sck rising edges, MSB-first shift register
- **TX**: Rising edge loads word, falling edge shifts out MSB

## DSP Module Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| WIDTH | 16 | Default data width for multipliers, I2S |
| PHASE_WIDTH | 32 | Phase accumulator width |
| OUTPUT_WIDTH | 16 | DDS sine/cosine output width |
| RMAX | 2 | CIC max decimation ratio |
| M | 1 | CIC differential delay |
| N | 2 | CIC filter order |
