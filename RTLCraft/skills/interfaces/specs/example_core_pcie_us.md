# example_core_pcie_us

## Parameters
- `AXIS_PCIE_DATA_WIDTH = 256`
- `AXIS_PCIE_KEEP_WIDTH = (AXIS_PCIE_DATA_WIDTH/32)`
- `AXIS_PCIE_RC_USER_WIDTH = AXIS_PCIE_DATA_WIDTH < 512 ? 75 : 161`
- `AXIS_PCIE_RQ_USER_WIDTH = AXIS_PCIE_DATA_WIDTH < 512 ? 60 : 137`
- `AXIS_PCIE_CQ_USER_WIDTH = AXIS_PCIE_DATA_WIDTH < 512 ? 85 : 183`
- `AXIS_PCIE_CC_USER_WIDTH = AXIS_PCIE_DATA_WIDTH < 512 ? 33 : 81`
- `RC_STRADDLE = AXIS_PCIE_DATA_WIDTH >= 256`
- `RQ_STRADDLE = AXIS_PCIE_DATA_WIDTH >= 512`
- `CQ_STRADDLE = AXIS_PCIE_DATA_WIDTH >= 512`
- `CC_STRADDLE = AXIS_PCIE_DATA_WIDTH >= 512`
- `RQ_SEQ_NUM_WIDTH = AXIS_PCIE_RQ_USER_WIDTH == 60 ? 4 : 6`
- `RQ_SEQ_NUM_ENABLE = 1`
- `IMM_ENABLE = 1`
- `IMM_WIDTH = 32`
- `PCIE_TAG_COUNT = 256`
- `READ_OP_TABLE_SIZE = PCIE_TAG_COUNT`
- `READ_TX_LIMIT = 2**(RQ_SEQ_NUM_WIDTH-1)`
- `READ_CPLH_FC_LIMIT = AXIS_PCIE_RQ_USER_WIDTH == 60 ? 64 : 256`
- `READ_CPLD_FC_LIMIT = AXIS_PCIE_RQ_USER_WIDTH == 60 ? 1024-64 : 2048-256`
- `WRITE_OP_TABLE_SIZE = 2**(RQ_SEQ_NUM_WIDTH-1)`
- `WRITE_TX_LIMIT = 2**(RQ_SEQ_NUM_WIDTH-1)`
- `BAR0_APERTURE = 24`
- `BAR2_APERTURE = 24`
- `BAR4_APERTURE = 16`
- `TLP_DATA_WIDTH = AXIS_PCIE_DATA_WIDTH`
- `TLP_STRB_WIDTH = TLP_DATA_WIDTH/32`
- `TLP_HDR_WIDTH = 128`
- `TLP_SEG_COUNT = 1`
- `TX_SEQ_NUM_COUNT = AXIS_PCIE_DATA_WIDTH < 512 ? 1 : 2`
- `TX_SEQ_NUM_WIDTH = RQ_SEQ_NUM_WIDTH-1`
- `TX_SEQ_NUM_ENABLE = RQ_SEQ_NUM_ENABLE`
- `PF_COUNT = 1`
- `VF_COUNT = 0`
- `F_COUNT = PF_COUNT+VF_COUNT`
